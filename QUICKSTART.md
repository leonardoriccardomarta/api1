# ⚡ Setup in 2 Minuti!

## 🚀 Passo 1: Installa
```bash
npm install
```

## 🎯 Passo 2: Groq API Key (GRATIS!)

1. Vai su [console.groq.com](https://console.groq.com)
2. Crea account (GRATIS)
3. Crea un file `.env`:
```env
GROQ_API_KEY=la-tua-chiave-groq
```

## 🔥 Passo 3: Deploy!
```bash
vercel --prod
```

**FATTO! La tua API è LIVE! 🚀**
